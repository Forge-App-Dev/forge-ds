Decisão estratégica

Rota escolhida & o que "consumível" significa

O objetivo é usar o forge-ds ao evoluir o forge-app (Expo/React Native). "Consumível" = tokens de fonte única + conhecimento para IA/humano + componentes importáveis.

Níveis de "consumível"

  • Tokens de fonte única — o app puxa cor/espaço/tipo/motion do tokens.json (via tokens.rn.js).
  • Conhecimento p/ IA + humano — prompts, guidelines e ADRs orientam o que se constrói.
  • Componentes importáveis — import { Button } from "@forge/ds" em RN.

Caminho A — faseado

Agora: tokens.json como fonte única gerando o theme RN + DS como fonte de conhecimento. Médio prazo: convergir componentes para React Native + react-native-web — o app importa nativamente e o catálogo renderiza o mesmo componente real. Aproveita os componentes RN que já existem no app.
